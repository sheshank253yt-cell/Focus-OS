# Focus OS — Personal Productivity Dashboard

  ## Tech Stack
  - React 18 + TypeScript
  - Vite (build tool)
  - Tailwind CSS v4
  - Wouter (routing)
  - Framer Motion (animations)
  - Clerk (authentication)
  - Lucide React (icons)
  - shadcn/ui (UI components)
  - localStorage (per-user data)
  - Canvas API (analytics charts)

  ## Setup

  1. Install dependencies:
     pnpm install   (or: npm install)

  2. Create a .env file with your Clerk key:
     VITE_CLERK_PUBLISHABLE_KEY=pk_test_...

  3. Run dev server:
     pnpm dev

  ## Modules
  - Home Dashboard
  - Work Scheduler (Kanban/List/Calendar + Pomodoro)
  - AI Tools Hub
  - Learning Roadmap
  - Analytics (Canvas API charts)
  - Notes (Sticky + Journal)
  - Settings
